Lythrum

Lythrum is a multiplayer third person shooter built with Unreal Engine 5 and C++. The game leverages network replication for seamless online gameplay, Behavior Trees and EQS for dynamic AI behavior alongside other key gameplay mechanics that enhance the cooperative combat experience across diverse battlefields.


Myrica

Myrica is a 3D top-down platformer where you play as a fairy spirit on a quest to restore a sacred tree and save the land. This RPG-inspired adventure challenges players to solve puzzles, collect power-ups, and battle relentless goblin hordes using powerful magic. Built with Unreal Engine 5 and C++, the game combines strategic exploration and an enchanting world filled with mystery and danger.


Thorpeus

Thorpeus is a VR first person shooter game featuring time-bending mechanics to deliver stylish and cinematic combat. Developed as my first VR project, Thorpeus was built using Unreal Engine 5 and C++ for Meta Quest. It features AI behaviors powered by Behavior Trees and EQS, destructible enemies using Chaos system, and key gameplay mechanics such as a weapon system, stagger effects, and unique abilities, ensuring dynamic and responsive VR combat.


SOFAni

SOFAni is an AR-powered mobile app designed to let users virtually place furniture in their homes with ease. As an academic end-of-year project, it provides an intuitive experience where users can browse a catalog, select items, and seamlessly visualize them in their space using augmented reality With AR mode, users can view their surroundings through their phone's camera and position 3D furniture models on any visible surface, adjusting their placement and rotation for a perfect fit. To enhance the experience, the app also features a guiding mascot to assist users along the way Developed in two versions, SOFAni was built using: Unity, C#, ARCore, Android SDK and Android Studio / Unreal Engine 5, C++, ARCore, and Android SDK


Android VS Corona

Android Vs Corona is a fast-paced arcade game created for an INSAT Android Club event following the pandemic. Designed to be both engaging and relevant to the times, it provided a fun and thematic experience for attendees. This project was built using Unity and C#.


Flappy Android

Flappy Android was created as part of a Unity instruction session designed for aspiring game developers among middle schoolers. This simple yet fun game serves as an accessible introduction to Unity, making it easy to dissect and understand the fundamentals of game development. This project was built with Unity and C#.
